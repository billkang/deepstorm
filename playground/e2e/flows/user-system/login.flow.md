## Flow: L01 - 正常登录

### 前置条件
- 应用已启动（`cd ../app && npm start`）
- 用户 `logintest_user` 已注册（密码 `test1234`）

### 执行步骤
1. 导航到登录页面 `http://localhost:3000/`
   ✅ 验证点：页面标题包含"登录"，登录表单可见
2. 在用户名输入框中输入 `logintest_user`
   ✅ 验证点：输入框内容正确
3. 在密码输入框中输入 `test1234`
   ✅ 验证点：输入框内容正确
4. 点击"登录"按钮
   ✅ 验证点：页面跳转到 `http://localhost:3000/dashboard.html`
5. 确认仪表盘加载
   ✅ 验证点：页面显示"任务管理系统"标题
   ✅ 验证点：页面右上角显示用户名 `logintest_user`
   ✅ 验证点：统计卡片显示"全部：0 待办：0 已完成：0"

### 环境要求
- 目标环境：test

## Flow: L02 - 错误密码登录

### 前置条件
- 应用已启动
- 用户 `logintest_user` 已注册（密码 `test1234`）

### 执行步骤
1. 导航到登录页面 `http://localhost:3000/`
   ✅ 验证点：登录表单可见
2. 输入用户名 `logintest_user`
3. 输入错误密码 `wrongpassword`
4. 点击"登录"按钮
   ✅ 验证点：页面不跳转，仍然停留在登录页
   ✅ 验证点：页面显示错误提示"用户名或密码错误"

### 环境要求
- 目标环境：test
