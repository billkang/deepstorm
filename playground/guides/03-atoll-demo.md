# Atoll 工具能力演示

> Atoll（环礁）是 DeepStorm 运维侧套件，提供部署辅助、监控查询、故障排查等运维能力。
> 当前 `atoll-ops` skill 已就绪，MCP Server 及深度集成功能在规划中。
> 实际操作请按 `playground/README.md` Phase 5 步骤执行。

---

## 使用方式

通过 CLI 安装 Atoll 后，在 Claude Code 中运行 `/atoll:atoll-ops` 触发运维工作流。

---

## 场景一：部署辅助

Atoll 将帮助运维人员：

- CI/CD 流水线管理（触发构建、查看状态）
- 多环境部署（开发 → 测试 → 预发布 → 生产）
- 版本回滚支持

## 场景二：监控查询

- 从监控平台（如 Prometheus、Grafana）查询指标
- 告警分析与响应建议
- 性能趋势查看

## 场景三：故障排查

- 日志查询与分析
- 服务状态检查
- 常见故障的排查 SOP
